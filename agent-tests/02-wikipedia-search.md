# Test 2: Wikipedia Search and Extract

## Objective
Test search functionality and content extraction on Wikipedia.

## Steps

1. Navigate to https://www.wikipedia.org
2. Use inspect to understand the page structure
3. Find the search input field and fill it with "Albert Einstein"
4. Click the search button or press Enter
5. Wait for the article page to load
6. Extract the article title (h1) and section headings (h2)
7. Take a screenshot of the article

## Expected Results
- Search input should accept text
- Navigation to article page should occur after search
- Query should return article title and headings
- Screenshot should capture the article

## Success Criteria
- Article title contains "Albert Einstein"
- Multiple section headings are extracted
- Page URL changes to the article URL
